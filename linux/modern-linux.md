# 모던 linux/unix 명령어

전통적인 CLI 명령어 >> 새로운 명령어 개발(대부분 배포판에 포함되지 않은 경우가 많다.)

대부분 Rust 언어로 개발된 이 CLI 모던 명령어들은 기본 리눅스 명령어보다 성능이 훨씬 좋고 colorized output을 제공해 시각적으로 좋아졌다.
